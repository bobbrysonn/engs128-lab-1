## Assignment Outline

1. Complete the **prelab** assignment to obtain the design specifications -- `README_Prelab.md`
	* A copy of the SSM2603 audio codec datasheet has been provided in this repository
	* You will also need to refer to the [Zybo technical reference manual](https://digilent.com/reference/programmable-logic/zybo-z7/reference-manual#audio) to find all the important information
2. Complete the **design tasks** in the assignment -- `README_Assignment.md`
	* Use the starter code provided in the `hw/` folder
	* When you get to Task 3, use the C code provided in the `sdk/` folder -- you will not need to modify the code for this assignment 
3. Give a live **demonstration** of your system during class (see Canvas for due date).
4. Submit your code and **post-lab report** to Canvas (see Canvas for due date).
	* Instructions on the assignment submission components can be found in the `README_Postlab.md` file.
	* Each student will submit their own report and code repository.
